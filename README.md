# Invoice System Frontend

React + TypeScript frontend for the Invoice System.

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- Vite

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`

- User authentication
- Dashboard with stats
